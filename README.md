# Launchy::Rails

Launches the browser automatically once the rails server has started up

## Installation

Add this line to your application's Gemfile:

    group :development do
        gem 'launchy-rails'
    end

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install launchy-rails
